# HRK Bio Polymers Website

Official website for HRK Bio Polymers - Global packaging solutions manufacturer specializing in durable, innovative, and eco-responsible polymer and bio-based packaging.

**Live Site:** [hrk.co.in](https://hrk.co.in)

## Overview

This is a modern, responsive website built with React + TypeScript + Tailwind CSS, showcasing HRK Bio Polymers' products, services, and company information.

## Features

- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI** - Built with Tailwind CSS for a professional appearance
- **Fast Navigation** - React Router for smooth page transitions
- **Optimized Performance** - Vite build tool for fast development and production builds
- **Accessible Components** - Lucide React icons for intuitive user interface

## Tech Stack

- **Frontend Framework:** React 19
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Routing:** React Router DOM
- **Build Tool:** Vite
- **Icons:** Lucide React
- **Deployment:** Vercel

## Getting Started

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/hrk-website.git
cd hrk-website
```

2. Install dependencies:
```bash
npm install
```

3. Start development server:
```bash
npm run dev
```

The site will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

The optimized build will be in the `dist/` folder.

### Preview Production Build

```bash
npm run preview
```

## Deployment

This project is configured for easy deployment on Vercel:

1. Push code to GitHub
2. Visit [vercel.com](https://vercel.com)
3. Import this repository
4. Vercel automatically detects Vite configuration
5. Deploy with one click
